The boot-loader is a short program used to burn the firmware to the microcontroller without any
programer device either like FLASH or volatile like RAM and jumps to the desired program
from there it take care of excution.

The process of buring the provided data to the program memory is controlled by the boot-loader.
A boot-loader is a program which will be the first thing to run and can load other application
into specific places in memory and is provided to the serial interface.
